# 页面结构稿 02-03

## 页面 02：智能管家追问页

### 页面目标

- 把“系统会自动分配体系”具象化。
- 不让用户感觉被打断，而是感觉被专业引导。

### 页面结构

1. 顶部
- 返回
- 管家标题
- 当前问题阶段标记

2. 原问题卡
- 展示用户刚才输入的问题

3. 管家判断卡
- 这是什么类型的问题
- 本轮建议主判体系
- 本轮建议辅助体系

4. 缺失条件卡
- 还差哪些信息
- 每项缺失信息用单独字段卡展示

5. 快速补充区
- 时间
- 地点
- 出生信息
- 户型图/照片

6. 底部按钮
- 继续补充并起算

### 文案原则

- 语气像顾问，不像报错框
- 不说“失败”
- 不说“格式错误”
- 说“还差这些信息，我再继续为你分配”

## 页面 03：风水工作台

### 页面目标

- 让用户第一次感觉到：我真的可以自己在手机上做风水判断。

### 页面结构

1. 顶部
- 返回
- 风水工作台标题
- 当前场景切换：住宅 / 办公室 / 店铺

2. 空间总览卡
- 地址
- 朝向
- 户型状态

3. 户型上传区
- 上传户型图
- 拍照上传
- 重新标注

4. 关键点位区
- 大门
- 主卧
- 灶位
- 办公桌 / 收银台

5. 系统分析区
- 当前格局结论
- 风险点
- 建议调整点
- 优先处理项

6. 底部操作
- 继续标注
- 重新分析
- 保存本次布局

### 视觉原则

- 比首页更工具化
- 仍然保留神秘质感
- 更强调结构、标注、判断
- 少一点宇宙感，多一点“玄学专业工作台”
