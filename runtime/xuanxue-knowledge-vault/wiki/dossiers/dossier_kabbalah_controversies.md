# 卡巴拉资料包：争议与流派差异

## 主要争议

犹太卡巴拉与西方秘传卡巴拉不是同一语境，宗教挪用和现代混合解释需要标注。

## 常见冲突类型

- 文本冲突：不同版本、不同注本的术语和规则不一致。
- 流派冲突：同一输入在不同流派下得出不同判断。
- 现代化冲突：当代咨询语言与传统断语目标不同。
- 验证冲突：案例选择偏差、事后解释和缺少失败样本。

## 本地推演处理

1. 先显示采用的流派口径。
2. 再列出可能产生不同结论的规则分歧。
3. 对争议高的判断自动降低置信度。
4. 不把市场流行说法包装成古法共识。
