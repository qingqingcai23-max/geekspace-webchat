# 八字资料包：关键人物与整理线索

## 徐子平

- 来源：[CText 数据页](https://ctext.org/datawiki.pl?if=en&res=432379)
- 当前可记录点：
  - 被后世视为子平法核心祖师形象。
  - 八字又被称为“子平八字”，说明其在传统叙事中的正统地位。

## 任铁樵

- 来源：[《滴天髓阐微》CText 条目](https://ctext.org/wiki.pl?if=en&remap=gb&res=221357)
- 当前可记录点：
  - 其评注是现代学习《滴天髓》时绕不开的解释层之一。
  - 适合放入“经典注释与后世重构层”。

## 现代观察入口：Berna Lee

- 来源：
  - [Chinese astrology short course page](https://sophia-project.net/lectures-and-seminars/shortcourses/2025-lee-chinese-cultural-a-and-a.php)
  - [UWTSD 论文库记录](https://repository.uwtsd.ac.uk/id/eprint/3390/)
- 当前可记录点：
  - 其研究更偏“中国占星与命理实践在现代社会中的世界观与实践”，不是单做八字文献学。
  - 适合作为“现代实践者如何理解中国命理”的观察层。

## 当前判断

- 八字体系的“学院派权威人物”不如道教研究和西方神秘学那样集中。
- 当前更适合按“古籍作者 / 注家 / 现代整理者 / 现代观察者”四层录入。
