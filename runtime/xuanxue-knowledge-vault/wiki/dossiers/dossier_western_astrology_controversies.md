# 西洋占星资料包：争议与问题清单

## 古典占星与现代心理占星

- 古典更强调技术、判断、时机与事件。
- 现代心理占星更强调人格、自我认识与成长叙事。

## 占星研究的学术位置

- [Sophia Centre](https://www.sophia-project.net/ma_caa.php) 与 [Nicholas Campion 教员页](https://www.uwtsd.ac.uk/staff/nicholas-campion) 表明学界确实研究占星。
- 但研究占星并不等于替占星断验作科学背书。

## 占星是否属于西方神秘学

- [UvA Western esotericism](https://www.uva.nl/en/discipline/religious-studies/western-esotericism/article.html) 明确把 astrology 放入 occult sciences。
- 但占星内部很多实践者会把自己与“神秘学”刻意区分。

## 技术分支差异

- 热带黄道与恒星黄道
- 宫位系统差异
- 卜卦、本命、择时、世运的边界

## 现代传播的简化问题

- 太阳星座化
- 社交媒体人格标签化
- 商业内容对古典技术细节的大幅裁剪
